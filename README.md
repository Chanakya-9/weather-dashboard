# 🌤️ Weather Dashboard

A modern weather application built with **React**, **Vite**, and **Material UI** that allows users to search for any city and view its current weather conditions using the OpenWeather API.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Display current temperature
- 🤗 Shows "Feels Like" temperature
- 💧 Humidity information
- 🌬️ Wind speed
- 📊 Atmospheric pressure
- 🌤️ Weather condition and description
- 🎨 Responsive and clean Material UI design

---

## 🛠️ Tech Stack

- React
- Vite
- Material UI (MUI)
- JavaScript (ES6+)
- CSS3
- OpenWeather API

---


## 🔑 Environment Variables

Create a `.env` file in the project root.

```env
VITE_WEATHER_API_KEY=YOUR_API_KEY
```

---

## 📷 API Used

OpenWeather Current Weather API

Example endpoint:

```text
https://api.openweathermap.org/data/2.5/weather?q={CITY_NAME}&appid={API_KEY}&units=metric
```
